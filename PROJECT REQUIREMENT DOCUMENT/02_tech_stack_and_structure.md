# S for Shopping - Product Requirements Document (PRD)
## Part 2: Tech Stack & Folder Structure

## 1. Tech Stack
The platform uses the following technologies:
- React + Vite
- Tailwind CSS
- Node.js
- Express.js
- Supabase (PostgreSQL, Auth, Storage)
- React Router
- Redux Toolkit
- Framer Motion
- Recharts
