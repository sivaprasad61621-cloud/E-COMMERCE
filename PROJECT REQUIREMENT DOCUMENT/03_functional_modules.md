# S for Shopping - Product Requirements Document (PRD)
## Part 3: Functional Modules

## 1. Modules Breakdown

### 1.1 Admin
- Login
- Dashboard
- Products CRUD
- Categories CRUD
- Orders
- Customers
- Reports
- Settings

### 1.2 Dashboard
- Total Products
- Total Orders
- Revenue
- Low Stock Alerts
- Sales Charts
- Recent Orders

### 1.3 Product Fields
Every product entity requires the following attributes:
- Name
- Description
- Category
- Price
- Discount
- SKU
- Stock
- Status
- Images

### 1.4 Orders
An order can progress through the following status workflow:
- Pending
- Packed
- Shipped
- Delivered
- Cancelled

### 1.5 Customers
- List
- Details
- Order History
